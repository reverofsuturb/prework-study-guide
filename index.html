Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements nested inside other elements are called child elements.
  • An element refers to the entire element from opening tag to closing tag.
  • A tag refers only to what appears inside the angled brackets.

CSS

  • CSS stands for Cascading Style Sheets.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files to the working branch

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • Variables are containers that store values, values can have different data types.
  • JavaScript is case sensitive.
  • Comments are snippets of text that can be added along with code, browsers ignore text marked as comments.
  • An operator is a mathematical sympol that produces a result based on two values (or variables).
  • Conditionals are code structures used to test if an expression returns true or not.
  • Functions are a way of packaging functionality that you wish to reuse. It's possible to define a body of code as a function that executes when you call the function name in your code.
  • Events are code structures that listen for activity in the browser, and run code in response.
